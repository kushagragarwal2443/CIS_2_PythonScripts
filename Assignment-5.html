Molecular Dynamics, these equations are used for each particle in every update step: $ \vec{F_{ext,i}} = \frac{d\vec{p_i}}{dt} = - \vec{\nabla}U $ and $ \vec{p_i} = m_i \frac{d\vec{r_i}}{dt} $

Lesson-4: Molecular Dynamics¶

  • Molecular Dynamics is essentially the application of the Newtons Second Law to molecules, where the force acting on each part of the molecule is computed by modelling its interaction with every other part of the system.
  • Total potential energy of the system: $$ U = \sum_{i\in\{bonds\}} \frac{1}{2} k_i (d_i - d_{0,i})^2 + \sum_{i\in\{angles\}} \frac{1}{2} k_i (\theta_i - \theta_{0,i})^2 + \sum_{i\in\{dihedrals\}} \phi ( cos(n\theta_i)-1) + \sum_{i,j\in\{non-bond~pairs\}} ( U_{LJ}(i,j) + U_{Columbic}(i,j)) $$
  • Equations of motion given by: $$ \frac{d\vec{p}_i}{dt} = \vec{F}_i \\ \frac{d \vec{r}_i}{dt} = \vec{p}_i/m_i $$
  • Euler Method: $$ \vec{p}_i(t+dt) = \vec{p}_i(t) + dt~\vec{F}_i(t) \\ \vec{r}_i(t+dt)=\vec{r}_i(t) + dt\vec{p}_i/m_i $$
  • Check on accuracy: constant total energy, total linear momentum
  • Example: Simple harmonic oscillator

Example: Argon¶

Argon atom is electrically neutral and a good model for the pair-wise interaction between two Argon atoms is given by Lennard-Jones potential, whose functional form is given by: $$ U_{LJ} (\vec{r}_i,\vec{r}_j) = 4 \epsilon \left( \left(\frac{\sigma}{r} \right)^{12}- \left(\frac{\sigma}{r} \right)^6 \right) $$ where $\vec{r} = \vec{r}_i - \vec{r}_j$ and $r=|\vec{r}|$. The force on particle $i$ due to its interaction via LJ potenital with particle $j$ is then given by: $$-\left( \frac{\partial U_{LJ}}{\partial x_i}, \frac{\partial U_{LJ}}{\partial y_i},\frac{\partial U_{LJ}}{\partial z_i} \right)$$ And the total potential of the system of $N$ Argon atoms is sum of all the pairwise interactions, i.e., $$ U = \frac{1}{2} \sum_{i,j=1, i\ne j}^N U_{LJ} (\vec{r}_i,\vec{r}_j) $$ Using the Euler method mentioned above, write a program to compute the trajectory of the system! Following functions might need to be written:

  • euler_integrate(timestep)

  • compute_force, compute_pairwiseLJ(i,j,sigma,epsilon)

class Atom:
        
    def __init__(self, r = np.zeros(3), p = np.zeros(3), m = 1.0,
                 sigma = 1.0, epsilon = 1.0, label='ATOM (unlabelled)'):
        self.r, self.p, self.m = r, p, m
        self.sigma, self.epsilon, self.label = sigma, epsilon, label
        assert type(self.r) is np.ndarray and len(self.r.shape) is 1 and self.r.shape[0] is 3
        assert type(self.p) is np.ndarray and len(self.p.shape) is 1 and self.p.shape[0] is 3
        assert type(self.sigma) is float and type(self.epsilon) is float and type(self.m) is float
        
    def __str__(self):
        return "Atom: " + str(self.label) + "\nPosition: " + str(self.r) + "\nMomentum: " + str(self.p)
    
    def force(self, atoms, v = lambda r, atom: fKX(r)):
        force = np.zeros(3)
        for atom in atoms:
            rad = self.r - atom.r
            force += (v(np.linalg.norm(rad), atom) * rad / np.linalg.norm(rad)) \
                        if np.linalg.norm(rad) != 0 else np.zeros(3)
        assert type(force) is np.ndarray and len(force.shape) is 1 and force.shape[0] is 3
        return force
    
    def update(self, atoms, dt = 0.001, v = lambda r, atom: fKX(r)):
        self.p += self.force(atoms, v) * dt
        self.r += self.p * dt / self.m
        assert type(self.r) is np.ndarray and len(self.r.shape) is 1 and self.r.shape[0] is 3
        return self.r
    
    def kineticEnergy(self):
        return 0.5 * np.linalg.norm(self.p)**2 / self.m
    
    def potentialEnergy(self, atoms, v = lambda r, atom: vKX(r)):
        total = 0
        for atom in atoms:
            total += v(np.linalg.norm(self.r - atom.r), atom)
        return total

def writeTrajectory(atoms = [], time = 0.0):
    with open('_coordinate.dat', 'w') as f:
        f.write('\n:: ++++++++++++++++++++ TIMESTEP = ' + str(time) + ' ++++++++++++++++++++\n')
        for atom in atoms:
            f.write(str(atom) + '\n')

Running the Simulation¶

The atoms are initialized with random positions, nil momenta, and 1.0 values for $\epsilon$ and $\sigma$. The Kinetic and Potential Energies are maintained. The final plot is the scatter of the final locations of the particles.

# Initialize all atoms
atoms = [Atom(r=np.random.random(3)*20, p=np.zeros(3), m=1.0, sigma=1.0, epsilon=1.0) for _ in range(100)]
kinetic, potential = np.array([]), np.array([])

x, y, z = [], [], []
# Update all atoms for many timesteps
for time in range(200):
    ke, pe = 0.0, 0.0
    for atom in atoms:
        atom.update(atoms, v = lambda r, atom: fLJ(r, atom), dt = 0.001)
        ke, pe = ke + atom.kineticEnergy(), pe + atom.potentialEnergy(atoms, v = lambda r, atom: vLJ(r, atom))
    kinetic, potential = np.append(kinetic, ke), np.append(potential, pe/2)
    if (time % 10 == 0):
        writeTrajectory()
    # Get the positions of each atom
    x.append([atom.r[0] for atom in atoms])
    y.append([atom.r[1] for atom in atoms])
    z.append([atom.r[2] for atom in atoms])
